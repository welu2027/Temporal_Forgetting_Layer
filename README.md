This project builds upon "[Temporal Sampling for Forgotten Reasoning in LLMs](https://arxiv.org/pdf/2505.20196)" by explaining the "why" behind temporal forgetting.

It investigates the mechanistic basis of temporal forgetting in LLMs, which is the phenomenon where models lose the ability to solve problems they previously solved correctly during RL training. Through activation patching, logit-lens analysis, and representational geometry, I localize which transformer layer is responsible for a math solution disappearing between training checkpoints. I find that forgetting is not distributed uniformly across the network, but concentrates in a critical layer range, suggesting that temporal forgetting reflects targeted overwriting of specific computational substructures rather than global weight drift.
